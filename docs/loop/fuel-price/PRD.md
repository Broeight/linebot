# PRD：油價查詢（台灣中油）

**版本**：1.0  
**日期**：2026-06-30  
**作者**：PM（閉環開發流程第一棒）  
**狀態**：草稿，待架構師接手

---

## 1. 背景與動機

家庭成員（台灣↔越南）每週固定詢問「這週油價多少」、「95 漲了嗎」——目前只能手動查中油官網。台灣中油採**浮動油價機制**，每週一早上 8:00 調整一次，使用者需求集中在週末前後。

本功能延續既有查詢服務模式（匯率、連假），只做「讀取＋顯示」，不涉及帳號、訂閱或推播。

---

## 2. 目標

- 使用者可用中文關鍵字或越南語自然問句，在 LINE 聊天視窗內查到台灣當週油價。
- 回覆要涵蓋家庭最常用的四個油品，並顯示生效日期，讓使用者判斷資訊是否最新。
- 不需 API 金鑰，符合本專案「免費優先」原則（資料來源選型留給架構師決定）。

---

## 3. 使用者故事

| ID | 角色 | 情境 | 期望結果 |
|----|------|------|----------|
| US-01 | 台灣家人 | 傳「油價」 | 取得四種油品現行牌價與生效日期 |
| US-02 | 台灣家人 | 傳「95油價」 | 只取得 95 無鉛汽油現行牌價 |
| US-03 | 台灣家人 | 傳「今天油價」 | 同 US-01，顯示本週牌價（強調「當週有效」） |
| US-04 | 越南家人 | 用越南語問「giá xăng hôm nay」（今天油價） | AI 工具觸發，取回摘要後用越南語回覆 |
| US-05 | 越南家人 | 用越南語問「giá dầu diesel」（柴油價格） | AI 工具觸發，只回超級柴油牌價 |
| US-06 | 任何成員 | 語音說「幾號油多少錢」 | 語音轉文字後 fallthrough 到 AI 工具或關鍵字路由，同樣得到油價回覆 |

## 4. 功能範圍

### 4.1 做（In Scope）

- **中文關鍵字指令**（走 `handler.js` 路由）
  - `油價`、`今天油價`、`本週油價` → 四種油品全部回覆
  - `92油價`、`95油價`、`98油價`、`柴油油價`（及常見別稱，見 §5.1）→ 單一油品回覆
  - `油價查詢` → 使用說明子選單

- **AI 工具支援**（走 `tools.js`，任何語言含越南語）
  - 偵測到「油價」意圖時呼叫 `get_fuel_price` 工具
  - 工具回傳英文摘要，AI 依使用者語言改寫後回覆

- **資料欄位**（需從資料來源取得下列欄位）
  - `product_name`：油品名稱（92 無鉛、95 無鉛、98 無鉛、超級柴油）
  - `price_ntd`：新台幣牌價（每公升，保留小數點後 1 位）
  - `effective_date`：本週牌價生效日期（YYYY-MM-DD，即當週週一）

- **快取**：記憶體快取，TTL 不超過 4 小時（資料每週一更新，快取不必 TTL 太短）

- **逾時防護**：對外 fetch 加 AbortController，逾時上限 8 秒

### 4.2 不做（Out of Scope，本期）

- 下週調漲/調降「預估值」（中油有時提早公布，但資料來源不穩定，留待下一期評估）
- 推播通知（每週一油價更新時主動通知）
- 歷史油價查詢
- 越南當地油價
- 台塑、速邁樂等非中油品牌

## 5. 功能需求

### 5.1 油品名稱對應表

| 使用者可能輸入 | 正規化油品 | 顯示名稱 |
|----------------|------------|----------|
| 92、92無鉛、九二 | UNLEADED_92 | 92 無鉛汽油 |
| 95、95無鉛、九五 | UNLEADED_95 | 95 無鉛汽油 |
| 98、98無鉛、九八 | UNLEADED_98 | 98 無鉛汽油 |
| 柴油、超柴、超級柴油 | SUPER_DIESEL | 超級柴油 |

越南語常見詞彙（AI 工具解讀，不需精確比對）：
- `xăng 92`、`xăng 95`、`xăng 98`：對應無鉛汽油各號
- `dầu diesel`、`dầu DO`：對應超級柴油
- `giá xăng`、`giá dầu`：不限油品 → 回傳全部四種

### 5.2 中文指令路由（`handler.js`）

```
/^(?:油價|今天油價|本週油價|這週油價|當週油價)$/
→ fuelPrice.lookup()（回傳全部四種）

/^(?:油價查詢)$/
→ fuelPrice.usage()（回傳使用說明）

/^(92|95|98|超柴|柴油|超級柴油|九二|九五|九八)[無鉛]?\s*油價$/
→ fuelPrice.lookup(productCode)（回傳單一油品）
```

優先序：油價指令在匯率之後、發票對獎之前插入（handler.js 約 150 行附近）。

### 5.3 AI 工具定義（`tools.js`）

新增工具 `get_fuel_price`，schema 如下（描述用中文，工具本體實作留給架構師）：

- **name**：`get_fuel_price`
- **description**：查詢台灣中油當週油價牌價。當使用者用任何語言（尤其越南語）詢問台灣油價、汽油價格、柴油多少錢時呼叫。
- **parameters**：
  - `product`（選填，string）：油品代碼（`UNLEADED_92` / `UNLEADED_95` / `UNLEADED_98` / `SUPER_DIESEL`），不填則回傳全部四種。

### 5.4 回覆格式（中文指令，`fuelPrice.lookup()`）

**全部四種**：
```
⛽ 台灣中油本週油價
生效日：YYYY-MM-DD（週一）

・92 無鉛汽油　X.X 元/公升
・95 無鉛汽油　X.X 元/公升
・98 無鉛汽油　X.X 元/公升
・超級柴油　　X.X 元/公升

資料來源：台灣中油（每週一調整）
```

**單一油品**：
```
⛽ 95 無鉛汽油：X.X 元/公升
生效日：YYYY-MM-DD（週一）

資料來源：台灣中油（每週一調整）
```

**取不到資料**：
```
目前無法取得油價資料，請稍後再試 🙏
```

### 5.5 服務層函式（`src/services/fuelPrice.js`，供架構師參照）

| 函式名稱 | 用途 | 回傳 |
|----------|------|------|
| `lookup(product?)` | 中文指令用，格式化中文字串 | `Promise<string>` |
| `usage()` | 使用說明子選單 | `string` |
| `getFuelPriceSummary(product?)` | AI 工具用，英文純文字摘要 | `Promise<string\|null>` |

`getFuelPriceSummary` 英文摘要範例（供架構師參考格式）：
```
Taiwan CPC fuel prices effective 2026-06-29:
Unleaded 92: 28.3 NTD/L; Unleaded 95: 29.8 NTD/L;
Unleaded 98: 32.0 NTD/L; Super Diesel: 27.5 NTD/L.
Source: Taiwan CPC (updated weekly on Monday). Reference prices only.
```

### 5.6 `timeContext()` 更新

在 `tools.js` 的 `timeContext()` 字串中補充提示：
> 「若使用者用任何語言詢問台灣油價、汽油、柴油，就呼叫 get_fuel_price 工具。」

## 7. 邊界與風險

### 7.1 資料來源風險

| 風險 | 說明 | 緩解方向 |
|------|------|----------|
| 無乾淨的免金鑰 API | 中油官網目前沒有公開 REST API；架構師需評估 HTML 爬蟲（CPC 網站）或第三方整合資料站的可行性與穩定性 | 由架構師決定，PRD 只定義需要的欄位（§4.1） |
| 每週一調整，資料延遲 | 若快取 TTL 過長，週一早上 8 點後用戶查到的可能仍是上週舊價 | 建議快取 TTL ≤ 4 小時，並在回覆中顯示生效日期讓用戶自行判斷 |
| 資料來源網站改版 | 爬蟲或第三方站點可能因版面改動而失效 | 架構師需評估備援方案或改版偵測機制（本期不強制要求） |
| 油品名稱對應不一致 | 各來源的油品命名可能與 §5.1 對應表有出入 | 架構師實作時需驗證欄位對應，必要時更新對應表 |

### 7.2 使用者體驗邊界

- **油價每公升格式**：顯示小數點後 1 位（例如 29.8 元），與中油官網一致。
- **調漲/調降方向**：本期不顯示漲跌箭頭或預估，避免資料來源不穩定造成誤導。
- **非中油品牌**：回覆底部可加一行「以台灣中油牌價為準，其他品牌可能略有差異」，由架構師決定是否加入。
- **越南語 fallthrough**：越南語自然問句走 AI 工具路徑，不走 `handler.js` 關鍵字路由。

### 7.3 範圍外確認（本期不做）

- 預估下週調漲/調降（中油浮動公告，資料來源不穩）
- 推播（每週一更新時主動通知，列入後期 backlog）
- 越南本地油價（不在台灣中油資料範圍內）
